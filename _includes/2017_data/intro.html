About this Event

Functional Scala 2019 (London Edition) is a free 2-day conference devoted to functional programming in Scala.

NOTE: You MUST have a ticket to attend. Tickets are completely FREE, although if you can afford, we ask you to chip in to help cover the costs for the venue!

Functional Scala 2019 (London Edition) is a free 2-day conference devoted to using the powerful features and libraries of the Scala programming language to build robust, modern, software that is easier to understand and maintain.

At this conference, you can find talks, workshops, and panels from leading open source contributors, commercial developers, and researchers from academia.

Agenda and speakers to be announced on an ongoing basis.
